# Safari Private Tours

Luxury safari booking platform for Kenya's premier wildlife experiences.

## Features

- **Tour Packages** – Curated safari itineraries with booking management
- **Vehicle Fleet** – Premium 4x4 safari vehicles with availability tracking
- **Photo Gallery** – Safari moments with category filtering and lightbox
- **Client Testimonials** – Review collection and display
- **Admin Dashboard** – Full CRUD management for all content
- **WhatsApp Integration** – Direct booking inquiries via WhatsApp

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React, TypeScript, Tailwind CSS, shadcn/ui, Framer Motion |
| Backend | Express.js, Node.js |
| Database | PostgreSQL, Drizzle ORM |
| Auth | Custom session-based |
| File Upload | Multer (local storage) |
